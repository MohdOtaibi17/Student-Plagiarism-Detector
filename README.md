# üìù Student Plagiarism Detector (NLP)

This project is a **Streamlit-based application** to detect plagiarism or similarity in student answers.  
It allows you to upload answers from **CSV, Excel, PDF, Word** files or enter them manually.  
The app then compares answers using multiple NLP models and visualizes the similarity using **heatmaps** and **network graphs**.

---

## üöÄ Features
- Upload answers from CSV, Excel, PDF, or Word files.
- Enter answers manually in the interface.
- Multiple similarity models:
  - TF-IDF
  - Sentence-BERT
  - Bag of Words
  - Jaccard Similarity
  - Word2Vec (Gensim)
- Interactive visualizations:
  - Similarity Heatmap
  - Student Relationship Graph
- Export results as CSV file.
